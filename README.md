# AI Content Management System (Frontend)

An enterprise-style **AI-assisted Content Management System (CMS)** built using **Angular**, designed with role-based access for **Admin** and **Author** users.
The application focuses on clean UI/UX, modular architecture, and real-world CMS workflows.
